Aplicação em JavaScript que calcula a média das notas de atletas em uma competição de ginástica artística.
Cada atleta recebe cinco notas, sendo descartadas a maior e a menor. A média é calculada com as três notas restantes.

funcionamento:
Percorre a lista de atletas com for

Ordena as notas em ordem crescente

Usa if para ignorar a maior e a menor nota

Calcula a média das três notas válidas

Exibe o nome, notas e média no console
